Note: we use Python 3.6

To install necessary requirements, type 'make setup'
To run tests, type 'make test'
To run local tournament, type 'make tournament'

To connect our NetworkPlayer to the Tournament server, run cd into src and run 'python networkAdministrator [name-of-player] [host] [port]'
To run our tournament server, cd into src and run 'python networkedGame [port]'
To run test-play-a-turn, use the play-a-turn.sh script at the root directory
